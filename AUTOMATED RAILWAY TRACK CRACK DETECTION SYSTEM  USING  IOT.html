AUTOMATED RAILWAY TRACK CRACK DETECTION SYSTEM USING IOT

The Automated Railway Track Crack Detection System Using IoT

is designed to automatically detect cracks or faults in railway tracks using sensors and send real-time alerts through the Internet of Things (IoT) to prevent accidents.

This system ensures train safety, reduces human error, and saves maintenance time by continuously monitoring the condition of the tracks.

Sensors Used:

✅ Ultrasonic Sensor Detects gaps/cracks by measuring distance variations

✅ Vibration Sensor Detects abnormal vibrations caused by cracks

✅ IR Sensor Detects discontinuity in track surface

✅ GPS Module Sends exact crack location

Working Principle:

Sensors continuously scan railway track Data sent to microcontroller

System checks thresholds: Distance variation → crack Abnormal vibration → structural issue

If detected: Send alert (SMS / Cloud dashboard) Include GPS location

🧾 Sample Arduino Code (Basic):

#define trigPin 9

#define echoPin 10

>long duration;

int distance;

void setup() {

Serial.begin(9600);

pinMode(trigPin, OUTPUT);

pinMode(echoPin, INPUT);

}

void loop() {

digitalWrite(trigPin, LOW);

delayMicroseconds(2);

digitalWrite(trigPin, HIGH);

delayMicroseconds(10);

digitalWrite(trigPin, LOW);

duration = pulseIn(echoPin, HIGH);

distance = duration * 0.034 / 2;

Serial.print("Distance: ");

Serial.println(distance);

if(distance > 15) {

Serial.println("Crack Detected!");

}

delay(500);

}
