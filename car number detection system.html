Car Number Detection System

Car Number Detection System in Python automatically detects and reads vehicle license plates using OpenCV.Car Number Detection System

Typical system has 3 main steps:

1. Image Input Camera / video feed / image file

2. License Plate Detection Detect the plate region using: OpenCV (traditional) Deep Learning (YOLO, Haar cascades)

3. Character Recognition (OCR) Extract text using: Tesseract OCR (pytesseract) Deep learning OCR models

Install Required Libraries:

pip install opencv-python pytesseract numpy imutils
